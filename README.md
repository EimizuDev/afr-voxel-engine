# AFR Engine
### A voxel game engine (C++ / Vulkan)

<Img src="docs/gifs/voxels.gif" style="height: 50vh">

## Overview

This repository contains a curated snapshot of the engine.

The game source, which leverages this engine, is private, although the demos are in the Releases.

## Features

- Voxel traversal: 3D DDA algorithm for fast ray traversal.
- Lighting: basic per-pixel lighting.

## What is excluded

- All utility batch files.
- Personal code comments and todos.

## Demo

Check out the [Releases](https://github.com/EimizuDev/afr-voxel-engine/releases). I'll release demos there showcasing new things added and my progress on the game I'm working on.

## Tech Stack

### Languages

C++, Slang, Lua.

### APIs / Libraries

Vulkan, Premake, entt, glfw, glm, spdlog, vk-bootstrap.
